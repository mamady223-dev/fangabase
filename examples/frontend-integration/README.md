# Exemple technique sans design

Cet exemple se limite au client `createFrontendClient` de `@fangabase/contracts`. Configurez une base se terminant par `/api/`, fournissez le cookie `fangabase_csrf` via la fonction `csrfToken`, puis utilisez `login`, `logout`, `organizations`, `billingSummary` ou `entitlements`.

Il ne contient aucune page, couleur, composant visuel ou clé privée. Il est facultatif et ne réactive aucune page de `examples/frontend-pages`.
