# Exemples de pages frontend

Ces pages sont des démonstrations fonctionnelles facultatives. Elles sont remplaçables, ne sont pas incluses dans l’application active et ne constituent ni un thème, ni une palette, ni une identité visuelle officielle FangaBase.

Le dossier `app` conserve les cinq exemples issus des jalons précédents : landing page, dashboard, tarification, facturation utilisateur et facturation administrative. Les exemples de facturation conservent leurs appels vers les proxies `/api/backend`. Pour les réutiliser, un étudiant choisit explicitement les fichiers utiles et les adapte à son propre design.

FangaBase reste headless par défaut. Copier ces exemples est toujours facultatif.
